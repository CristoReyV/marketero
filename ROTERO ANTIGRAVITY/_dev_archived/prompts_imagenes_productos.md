# Prompts para Imágenes del Catálogo B2B Rotero

Aquí tienes los 12 prompts específicos por producto. Te sugiero usar tu propia cuenta de ChatGPT Plus (DALL-E 3), Midjourney o alguna herramienta similar para generarlos y descargarlos. 

Una vez los tengas, **guárdalos en la carpeta `assets/tienda/`** utilizando el **Nombre de archivo sugerido**. Esto facilitará nuestra labor a la hora de actualizar la base de datos automáticamente.
